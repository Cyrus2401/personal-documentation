# 🔧 Extensions VS Code Essentielles

## 📝 Édition & Formatage

### Auto Close Tag
Ferme automatiquement les balises HTML/XML lorsque vous tapez la balise d'ouverture.

### Auto Rename Tag
Renomme automatiquement la balise de fermeture quand vous modifiez la balise d'ouverture.

### Better Comments
Colore les commentaires selon leur type (TODO, FIXME, etc.) pour mieux les repérer.

### Bracket Pair Color
Attribue une couleur différente à chaque niveau de parenthèses et crochets.

### CodeSnap
Crée de belles captures d'écran de votre code avec coloration syntaxique.

### Colorize
Colore automatiquement les codes couleur dans votre code CSS/HTML.

### Database Client JDBC
Client universel pour se connecter et gérer différentes bases de données (MySQL, PostgreSQL, etc.).

### DotENV
Ajoute la coloration syntaxique et la validation pour les fichiers .env.

### Error Lens
Affiche les erreurs et avertissements directement sur la ligne concernée, sans avoir à survoler le code.

### French Language Pack for Visual Studio Code
Traduit l'interface de VS Code en français.

### GitLens — Git supercharged
Affiche l'historique Git, les auteurs de chaque ligne et améliore la navigation dans le code.

### HTML CSS Support
Fournit l'autocomplétion CSS directement dans vos fichiers HTML.

### IntelliSense for CSS class names in HTML
Suggère automatiquement les noms de classes CSS existantes dans votre HTML.

### JavaScript (ES6) code snippets
Propose des raccourcis pour écrire rapidement du code JavaScript moderne.

### JavaScript and TypeScript Nightly
Donne accès aux dernières fonctionnalités JavaScript et TypeScript avant leur sortie officielle.

### jQuery Code Snippets
Fournit des raccourcis pour écrire du code jQuery plus rapidement.

### Laravel Artisan
Permet d'exécuter les commandes Artisan de Laravel directement depuis VS Code.

### Laravel Blade formatter
Formate automatiquement vos fichiers de templates Blade Laravel.

### Laravel Blade Snippets
Propose des raccourcis pour les directives et structures Blade.

### Laravel Blade Spacer
Ajoute automatiquement des espaces appropriés dans vos fichiers Blade.

### Laravel Extra Intellisense
Suggère automatiquement les routes, vues et configurations Laravel.

### Live Code Metrics
Calcule la complexité de votre code et affiche des métriques pour améliorer la qualité.

### Live Server
Lance un serveur local qui recharge automatiquement la page à chaque modification.

### Live Share
Permet de coder en collaboration en temps réel avec d'autres développeurs.

### Markdown All in One
Offre tous les outils nécessaires pour écrire en Markdown (raccourcis, prévisualisation, etc.).

### Material Icon Theme
Remplace les icônes par défaut par des icônes Material Design plus modernes.

### Path Intellisense
Complète automatiquement les chemins de fichiers lors de l'importation.

### PHP Debug
Permet de déboguer du code PHP avec des points d'arrêt et l'inspection de variables.

### PHP Extension Pack
Regroupe plusieurs extensions essentielles pour le développement PHP.

### PHP Intelephense
Fournit une autocomplétion avancée et une navigation intelligente pour PHP.

### PHP IntelliSense
Offre une aide à la saisie et des suggestions pour le code PHP.

### PHP Namespace Resolver
Importe et résout automatiquement les namespaces PHP.

### PostgreSQL
Ajoute des outils pour travailler avec la base de données PostgreSQL.

### Postman
Permet de tester vos APIs REST directement depuis VS Code.

### Prettier - Code formatter
Formate automatiquement votre code pour maintenir un style cohérent.

### SCSS IntelliSense
Fournit l'autocomplétion pour les variables, mixins et fonctions Sass/SCSS.

### SFTP
Synchronise vos fichiers locaux avec un serveur distant via SFTP/FTP.

### Symfony code snippets
Propose des raccourcis pour écrire du code Symfony plus rapidement.

### Symfony for VSCode
Ajoute des outils et fonctionnalités spécifiques au framework Symfony.

### Tailwind CSS IntelliSense
Suggère automatiquement les classes Tailwind CSS avec leur aperçu.

### Thunder Client
Client API REST léger pour tester vos endpoints rapidement.

### Twig code snippets
Fournit des raccourcis pour écrire des templates Twig.

### Twig Language 2
Ajoute la coloration syntaxique pour les fichiers de templates Twig.
